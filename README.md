## 初始化依赖
```bash
tnpm run bootstrap
```

## 发布

获取 token 解决 lerna 发布 tnpm 二次验证的问题

```bash
tnpm run otp:release
```

## 单元测试

关于[单元测试](https://yuque.antfin-inc.com/oasis3d/mlxz18/fhvrag)

```bash
tnpm run test
```

### 测试覆盖率

```bash
tnpm run test-cov
```

查看 coverage/lcov-report/index.html
